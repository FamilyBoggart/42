
1. access(): Access comprueba si el programa puede acceder al pathname dado
	pathname: path
	mode: R_OK, W_OK, X_OK, y F_OK si existe
	retorna 0 si esta OK y -1 si es KO

2. dup2(int oldfd, int newfd):  duplica el filedescriptor del archivo (oldfd) en uno dado (newfd), apuntando ambos al mismo archivo
	si newfd y oldfd son el mismo, la funcion retornará -1
	si newfd esta abierto, dup2 lo cerrará primero antes de crear una copia.
	si olfd es -1 (no existe), dup2 creará un newfd libre asociado a ese archivo

3. fork(). Sirve para crear un nuevo proceso, el proceso padre crea un proceso hijo
	el proceso padre tiene un id (pid). El proceso hijo lo hereda en (ppid)
	el fork devuelve 0 si se trata del proceso hijo.
	Fork tambien traspasa los file descriptors
